# Review Format

Use this structure when reviewing collaborator outputs.

## Sections

1. `审查结论`
2. `一句话结论`
3. `关键结果`
4. `同意的部分`
5. `不同意或保留意见的部分`
6. `是否允许写入正式基线`
7. `下一步建议`

## Review checks

Always verify:

1. did the collaborator answer the intended question
2. did they stay within scope
3. is the evidence sufficient
4. can the result be promoted to baseline
5. is the next round still worth doing

## Decision levels

- `通过`
- `部分通过`
- `不通过`

Use `部分通过` when the work is useful but does not fully answer the intended line.
